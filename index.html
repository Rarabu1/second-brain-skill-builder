How to use this

Linear if you're new. Skip around if you're not.

The four modules are self-contained. Do them in order, or jump to the one that matches where you are. Each one ends with something you can actually use.

If you've never opened Terminal

Start with the warmup.

Twenty minutes in a safe sandbox removes the only barrier most knowledge workers ever hit with the setup guide.

If you live in a terminal already

Skip straight to the setup.

Chapter 1 takes about ten minutes. By chapter 3 you'll have a working morning triage routine reading your real email and chat.

If you want your files now

Use the personalizer.

Skip the guide. Answer five questions and get a CLAUDE.md, soul.md, morning brief, and project files generated to your exact context.

If you're running a training

Send people here.

One URL covers all four modules. Have attendees start with the warmup, then personalize before the setup guide — it makes the system concrete.

If your prompts keep missing the mark

Go to the prompt guide.

The gap between a good prompt and a vague one is usually one of six layers. The framework takes fifteen minutes to learn. The Brief builds one for your next task.
